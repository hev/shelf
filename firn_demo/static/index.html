shelf · search on object storage

The book search of shelf, running directly on Firn — vector, BM25, and hybrid search over object storage (MinIO/S3), with a RAM+NVMe cache in front. Firn doesn't route, so this app picks the strategy by query length: a word or two → keyword (BM25), a few words → fused (BM25 ⊕ vector by RRF), a sentence → semantic (ANN over a bge-small embedding). The headline here is the cache: run a query, then run it again and watch it return warm with zero backend requests.
